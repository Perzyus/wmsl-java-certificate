Question 16

Which statement is true about java.util.stream.Stream?

A. A stream cannot be consumed more than once.
B. The execution mode of streams can be changed during processing.
C. Streams are intended to modify the source data.
D. A parallel stream is always faster than an equivalent sequential stream.

Answer: B


-----------------------------------
P. note

A: Consume ไม่สามารถ เอาไปเรียก method อื่นๆ ได้
B: การทำงานของ stream สามารถเปลี่ยน mode ได้ตลอดสาย
C: stream ตั้งใจจะเข้าไปแก้ไขตัวตั้งต้น (Collections)
D: parallel stream เร็วกว่า stream แบบธรรมดา 

--> A parallel stream: <collection>.parallelStream()
--> A sequential stream: <collection>.stream().parallel()
